App Bar

```jsx
import AppBar, { Button } from "./AppBar";

<AppBar>
  <Button>Home</Button>
  <Button>About</Button>

```

App Bar with custom props

```jsx
import AppBar, { Button } from "./AppBar";

<AppBar backgroundColor="#343a40" color="white">
  <Button>Home</Button>
  <Button>About</Button>
  <Button>Contact</Button>
</AppBar>;
```
